Torque Integrated Development Editor (torque-ide) is an open-source, cross-platform editor for the Torque Game Engine.

The torque-ide uses wxWidgets for the cross-platform GUI library and non platform specific C++ to allow for maximum cross-platform ability.

It can be used for Torque script files as well as Tribes/Tribes2 game scripts...

There are plans to implement Python interoperability to allow for "plugins".
 
The torque-ide (for Windows) requires nothing extra to be installed in addition to the installer. The *nix version of the torque-ide requires GTK. Mac requirements are unkown at this time.
